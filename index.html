Baguette Bot Documentation

A documentation for multifunctional moderation, utility and role management bot called Baguette. The prefix can be changed using the 'prefix' command. Let's consider that the prefix is '!'.

!help

Brings you to the help page.

!prefix <new prefix>

Sets the bot prefix to whatever you want it to be.

!ban <@mention> [reason]

Instantly bans a member that you mentioned from the server.

!kick <@mention> [reason]

Instantly kicks a member that you mentioned from the server.

!mute || !stfu <@Mention> <timespan>

Instantly mutes a member that you mentioned for a specific amount of time.

!purge <amount>

Deletes a specific amount of messages.

!serverinfo

Shows an information about current server (creation time, ID, channels, region, owner)

!snipe

Shows the last deleted message in current channel.

!whois [@mention]

Shows an information about specific user (when joined, when created, ID, roles)

!avatar [@mention]

Shows a profile picture of a specific user.

!afk <reason>

Sends a message every time someone pings you.

!nuke

Copies the channel and deletes it to make it clean and new.

!invfilter <on/off>

Deletes all the invite links sent in the server.

!soon
